## Installation

Make sure to have python3 installed in your desired terminal

For equivalent result use the same tools as listed

* A running VM with Ubuntu installed
* The files provided
* Correct command usage

## How to run the bot
Server

```
python3 server.py 
```
Client
```
python3 client2.py 127.0.0.1 8888
```
